
## Nuxt Mdi module

![Nuxt Mdi Website](/images/screenshots/nuxt-mdi-website.png)

This design system uses the [Nuxt Mdi module](https://nuxt-mdi.emilrosenius.com/){:target="_blank"}.

It provides access to the full [Material Design Icons](https://pictogrammers.com/library/mdi/){:target="_blank"} set using a simple and flexible Vue component.


### Example
```vue
<template>  
    <MdiIcon icon="mdiAccount" />
</template>
```

### Usage

Please have a look at how to use the Nuxt Mdi module in the [module documentation](https://nuxt-mdi.emilrosenius.com/introduction/usage){:target="_blank"}.
